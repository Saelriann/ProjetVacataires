Bonjour ! 
Ici Robin et Marie Anne. Voici notre projet concernant les vacataires de l'UHA. Il est en cours de construction... 

-------------------------------------------------------------
De quoi avez-vous besoin pour faire fonctionner ce site ...? 
-------------------------------------------------------------
	- Apache Web Server
	- MySQL version 5.0 ou ultérieure
	- PHP version 5.3 ou ultérieure
-------------------------------------------------------------

------------------------------
Afin de l'utiliser localement
------------------------------
	1. Copiez les fichiers sur votre serveur.
	2. Créez une base de données et importez projet_vacataires.sql.
	3. Modifiez le fichier de configuration library/config.php en remplissant la configuration voulue (pensez à modifier le mot de passe, le nom du répertoire, etc.).
	4. Tadam!


-------------------
Déjà présent en BDD
-------------------
	Certaines données sont déjà présentes en base de données. Comme vous pouvez le constater, le mot de passe est crypté en base. Comme il s'agit d'un jeu de test, le mot de passe est simplement le prénom suivi de 123*
	ex : jessica.blyet@uha.fr = jessica123*
	
	Par défaut, les comptes créés sont les vacataires. Essayez de créer le vôtre !
